# Climate Control India - Admin Dashboard

Administrative dashboard to manage product catalogs and inquiries for Climate Control Systems India.

## Folder Structure
- `admin/` - React + Vite frontend
- `backend/` - Node.js + Express API backend

## Local Setup

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd admin
npm install
npm run dev
```
